
### Database Design Decisions:

#### `users` Table:
- **Purpose:** Stores admin credentials for authentication
- **Why VARCHAR(255) for email?** Accommodates long email addresses
- **Why UNIQUE on email?** Prevents duplicate admin accounts
- **Why default role 'admin'?** Simplifies user creation
- **Security Note:** Passwords stored in plain text (not recommended for production)

#### `issues` Table:
- **Primary Key:** Auto-incrementing ID for unique issue identification
- **Category:** Predefined types (Pothole, Garbage, Streetlight, etc.)
- **Description:** TEXT field for detailed issue descriptions
- **Location:** Geographic location where issue occurs
- **Status:** Workflow states (Pending → In Progress → Resolved)
- **Image Paths:** Store file paths for uploaded images
- **Timestamps:** Track creation and modification times
- **Why separate proof paths?** Allows tracking work start and completion separately

### Database Relationships:
- **No foreign keys:** Simple design, all issues are independent
- **Why no user-issue relationship?** Citizens are anonymous, only admins are tracked

## 🔌 DATABASE CONNECTION MANAGEMENT (DBConnection.java)

```java
package app.db;

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;

public class DBConnection {

    private static final String DB_URL = "jdbc:mysql://localhost:3306/civic_app?useSSL=false&allowPublicKeyRetrieval=true&serverTimezone=UTC";
    private static final String DB_USER = "civic_user";
    private static final String DB_PASSWORD = "1234";

    private static Connection connection;

    private DBConnection() {}

    public static Connection getConnection() throws SQLException {
        if (connection == null || connection.isClosed()) {
            try {
                Class.forName("com.mysql.cj.jdbc.Driver");
                connection = DriverManager.getConnection(DB_URL, DB_USER, DB_PASSWORD);
            } catch (ClassNotFoundException e) {
                throw new SQLException("MySQL JDBC driver not found.", e);
            } catch (SQLException e) {
                throw new SQLException("Unable to connect to MySQL. Check JDBC settings.", e);
            }
        }
        return connection;
    }
}
```

### Line-by-Line Analysis:

1. **`package app.db;`** - Database layer package
2. **Imports:** Standard JDBC classes for database connectivity
3. **`public class DBConnection {`** - Singleton pattern class
4. **Constants:** Database configuration (hardcoded for simplicity)
   - **DB_URL:** MySQL connection string with timezone settings
   - **DB_USER/DB_PASSWORD:** Database credentials
5. **`private static Connection connection;`** - Singleton connection instance
6. **`private DBConnection() {}`** - Private constructor prevents instantiation
7. **`public static Connection getConnection() throws SQLException {`** - Public accessor method
8. **Connection Check:** Only create new connection if null or closed
9. **`Class.forName("com.mysql.cj.jdbc.Driver");`** - Load MySQL JDBC driver
10. **`connection = DriverManager.getConnection(DB_URL, DB_USER, DB_PASSWORD);`** - Establish connection
11. **Exception Handling:** Specific error messages for troubleshooting
12. **`return connection;`** - Return the connection instance

### Design Patterns Used:
- **Singleton Pattern:** Ensures only one database connection exists
- **Lazy Initialization:** Connection created only when first requested
- **Exception Propagation:** Throws SQLException for caller to handle

### Why This Implementation?
- **Connection Pooling Alternative:** Simple singleton for desktop app
- **Resource Management:** Reuses connection to avoid overhead
- **Error Clarity:** Specific error messages help with debugging
- **Security:** Credentials in code (not recommended for production)

### Connection URL Parameters:
- **`useSSL=false`:** Disables SSL for local development
- **`allowPublicKeyRetrieval=true`:** Allows key retrieval in some MySQL versions
- **`serverTimezone=UTC`:** Sets timezone to avoid timestamp issues

   - Initializes timestamps to current time

```java
public boolean addIssue(Issue issue) {
    String sql = "INSERT INTO issues (category, description, location, status, image_path) VALUES (?, ?, ?, ?, ?)";

    try (Connection conn = DBConnection.getConnection();
         PreparedStatement stmt = conn.prepareStatement(sql)) {

        stmt.setString(1, issue.getCategory());
        stmt.setString(2, issue.getDescription());
        stmt.setString(3, issue.getLocation());
        stmt.setString(4, issue.getStatus());
        stmt.setString(5, issue.getImagePath());

        return stmt.executeUpdate() == 1;

    } catch (SQLException ex) {
        ex.printStackTrace();
        return false;
    }
}
```
**Purpose:** Creates new civic issue in database
**Why PreparedStatement?** Prevents SQL injection attacks
**Return Value:** Boolean indicates success/failure
**Exception Handling:** Logs errors, returns false on failure

#### 5. `filterIssues(String location, String status, String category)`

```java
public List<Issue> filterIssues(String location, String status, String category) {
    StringBuilder sql = new StringBuilder("SELECT * FROM issues WHERE 1=1");
    List<Object> params = new ArrayList<>();

    if (location != null && !location.trim().isEmpty()) {
        sql.append(" AND location LIKE ?");
        params.add("%" + location + "%");
    }
    if (status != null && !status.trim().isEmpty()) {
        sql.append(" AND status = ?");
        params.add(status);
    }
    if (category != null && !category.trim().isEmpty()) {
        sql.append(" AND category LIKE ?");
        params.add("%" + category + "%");
    }

    sql.append(" ORDER BY created_at DESC");

    List<Issue> issues = new ArrayList<>();
    try (Connection conn = DBConnection.getConnection();
         PreparedStatement stmt = conn.prepareStatement(sql.toString())) {
        for (int i = 0; i < params.size(); i++) {
            stmt.setObject(i + 1, params.get(i));
        }
        try (ResultSet rs = stmt.executeQuery()) {
            while (rs.next()) {
                issues.add(mapIssue(rs));
            }
        }
    } catch (SQLException ex) {
        ex.printStackTrace();
    }
    return issues;
}
```
**Purpose:** Admin dashboard filtering (exact status match, partial category/location)
**Parameter List:** Cleaner parameter management than manual indexing

#### 6. `updateIssueStatus(int id, String newStatus, String proofImagePath)`
```java
public boolean updateIssueStatus(int id, String newStatus, String proofImagePath) {
    // First, get current issue to validate transition
    Issue currentIssue = getIssueById(id);
    if (currentIssue == null) {
        return false;
    }

    String currentStatus = currentIssue.getStatus();

    // Validate status transition
    if (!isValidStatusTransition(currentStatus, newStatus)) {
        return false;
    }

    // Validate proof image requirement
    if ((newStatus.equals("In Progress") || newStatus.equals("Resolved")) && (proofImagePath == null || proofImagePath.trim().isEmpty())) {
        return false;
    }

    String sql = "UPDATE issues SET status = ?, updated_at = CURRENT_TIMESTAMP";
    if (newStatus.equals("In Progress")) {
        sql += ", proof_start_path = ?";
    } else if (newStatus.equals("Resolved")) {
        sql += ", proof_end_path = ?";
    }
    sql += " WHERE id = ?";

    try (Connection conn = DBConnection.getConnection();
         PreparedStatement stmt = conn.prepareStatement(sql)) {
        stmt.setString(1, newStatus);
        if (newStatus.equals("In Progress")) {
            stmt.setString(2, proofImagePath);
            stmt.setInt(3, id);
        } else if (newStatus.equals("Resolved")) {
            stmt.setString(2, proofImagePath);
            stmt.setInt(3, id);
        } else {
            stmt.setInt(2, id);
        }
        return stmt.executeUpdate() == 1;
    } catch (SQLException ex) {
        ex.printStackTrace();
        return false;
    }
}
```
**Purpose:** Updates issue status with workflow validation
**Business Rules:** Enforces status transition logic and proof requirements
**Dynamic SQL:** Different columns updated based on status

#### 7. `isValidStatusTransition(String currentStatus, String newStatus)`
```java
private boolean isValidStatusTransition(String currentStatus, String newStatus) {
    switch (currentStatus) {
        case "Pending":
            return newStatus.equals("In Progress");
        case "In Progress":
            return newStatus.equals("Resolved");
        case "Resolved":
            return false; // Cannot change from Resolved
        default:
            return false;
    }
}
```
**Purpose:** Enforces business workflow rules
**State Machine:** Defines valid state transitions

```java
public Map<String, Integer> getDashboardStats(String location) {
    Map<String, Integer> stats = new HashMap<>();
    String sql;
    PreparedStatement stmt = null;

    if (location != null && !location.trim().isEmpty()) {
        sql = "SELECT status, COUNT(*) AS count FROM issues WHERE location LIKE ? GROUP BY status";
        try {
            stmt = DBConnection.getConnection().prepareStatement(sql);
            stmt.setString(1, "%" + location + "%");
        } catch (SQLException ex) {
            ex.printStackTrace();
            return stats;
        }
    } else {
        sql = "SELECT status, COUNT(*) AS count FROM issues GROUP BY status";
        try {
            stmt = DBConnection.getConnection().prepareStatement(sql);
        } catch (SQLException ex) {
            ex.printStackTrace();
            return stats;
        }
    }

    try (ResultSet rs = stmt.executeQuery()) {
        int total = 0;
        while (rs.next()) {
            String status = rs.getString("status");
            int count = rs.getInt("count");
            stats.put(status, count);
            total += count;
        }
        stats.put("Total", total);
    } catch (SQLException ex) {
        ex.printStackTrace();
    } finally {
        if (stmt != null) {
            try {
                stmt.close();
            } catch (SQLException ex) {
                ex.printStackTrace();
            }
        }
    }
    return stats;
}
```
**Purpose:** Generates statistics for dashboard cards
**Aggregation:** Groups issues by status and counts them
**Location Filtering:** Optional location-based stats
